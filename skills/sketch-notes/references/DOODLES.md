## Hand-drawn Arrow Styles

### Simple Arrows
```
↓  Down arrow (simple)
⬇️ Down arrow (filled)
↘️ Down-right arrow
➡️ Right arrow
⬅️ Left arrow
⬆️ Up arrow
```

### Decorative Connectors
```
~~→  Wavy arrow
===>  Double line
--->  Dashed
···>  Dotted
```

## Sketch Style Tips

### Hand-drawn Effect
- Use slight rotation (±2-3 degrees)
- Apply wobbly border-radius
- Add subtle shadow for depth
- Mix fonts (Caveat + Kalam)

### Authentic Touches
- Scatter doodles randomly
- Add coffee stain effects
- Use tape decorations
- Include handwritten annotations
- Apply pencil color variations

### Layout Suggestions
- Vertical flow with arrows
- Asymmetric positioning
- Overlapping elements slightly
- Natural spacing (not too perfect)
